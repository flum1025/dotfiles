# dotfiles

## Dependencies

for mac 
- brew

for Linux

- apt
- curl

## Support

- `bash`
- `zsh`
- `tmux`
- `vim 8.0+`
- `nvim`
- `iterm`

## Getting Started

    $ curl -fsSL https://raw.githubusercontent.com/flum1025/dotfiles/master/qinstall.sh | bash
