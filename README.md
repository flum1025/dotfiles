# dotfiles

## Dependencies

You should install following applications.

for MacOS

- [MacVim-KaoriYa](https://github.com/splhack/macvim-kaoriya)

for Linux

- apt
- curl

## Support

- `bash`
- `zsh`
- `tmux`
- `vim`
- `nvim`
- `iterm`
