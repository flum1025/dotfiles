# dotfiles

## Dependencies

You should install following applications.

for MacOS

- [MacVim-KaoriYa](https://github.com/splhack/macvim-kaoriya)

for Linux

- apt
- curl

## Support

- `bash`
- `zsh`
- `tmux`
- `vim`
- `nvim`
- `iterm`

## Getting Started

    $ curl -fsSL https://raw.githubusercontent.com/flum1025/dotfiles/master/qinstall.sh | sh
